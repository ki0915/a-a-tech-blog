정보 처리 설비

  • IPF는 모든 컴퓨터 하드웨어, 소프트웨어, 데이터를 말한다.
  • 하드나 USB에서 보관되는 데이터들이다.

DBA (데이터 베이스 관리자)

- 데이터 베이스 관리는 감사인이 가장 중요하게 봐야 한다.

가장 중요하게 봐야 하는 이유는?

모든 데이터에 접근이 가능하니까

  • DBA는 직무 분리를 굉장히 빡세게 해야한다.
  • DBA의 작업 내용은 모두 경영진이 승인해야 한다.
  • 복구와 백업을 정의하고 그 책임은 DBA가 진다.
  • 갱신과 동시성 제어해서 무결성 관리한다.

라이브러리

  • 컴퓨터 안에 있는 모든 프로그램 제어나 버전을 관리한다.
개발 개발/테스트 라이브러리
운영 소스코드 / OBJ 목적코드 라이브 러리

중요 사항

- 품질 보증, 개발/설계/유지보수, 운영은 서로 겸직이 불가능하다.
- DBA/품질 관리자/보안 관리자, 개발/설계/유지보수, 운영은 서로 겸직이 불가능하다.

외주 업체 선정

  • 벤치마크 RFI를 통해서 괜찮은 업체들을 선정한다.
  • 괜찮은 업체가 있으면 RFP 를 통해서 제안서를 보낸다.
  • 외주 후보 업체가 보낸 제안서를 보고 어느 업체가 괜찮은지 선택한다.

* 한 업체만 선택하지 않아도 괜찮다
* 여러 업체 고르고 하나만 주업체로 골라도 괜찮다.

여러 가지 검증이나 테스트 방법

ITF(통합설비시설)
- 온라인 감사 시스템 감사 시에 사용
- 실제 운영 중인 컴퓨터에 테스트 값을 넣고 기대값이랑 비교한다.
- 온라인 감사 시스템 감사 시에 사용
- 실제 운영 중인 컴퓨터에 테스트 값을 넣고 기대값이랑 비교한다.

완전성 검사, 정확성 검사, 검사에 대응하는 통제

완전성 검사

  • 데이터가 누락되지는 않았는지 중복은 존재 하지 않는 지를 검사한다. 
    ** ex) 패러티 체크,  RUN TO RUN, 거래 로그, 배지 레지스터

정확성 검사 

  • 데이터가 정확한지를 판단하는 검사
    ** ex) 체크 디지트, 합리성 체크
